Необходимо разработать Telegram бота на python.

Общая задача: 

Telegram бот позволяет найти на сайте Aeroflot.ru билеты за бонусные мили.

В папку с проектом уже подгружен файл city_codes.py - проанализируй его и используй. Здесь указаны все пункта отправления и назначения.

При первом контакте (start) бот сообщает следующее: "Добро пожаловать в проект Milestrade. Я проверю наличие билетов за бонусные мили на нужную дату. Сообщу стоимость и количество доступных авиабилетов к оформлению"

Появляется кнопка "Поиск"

1. Бот запрашивает: Введите направление откуда летим
2. Пользователь вводит
3. Бот запрашивает: Введите направление куда летим
4. Пользователь вводит
5. Бот запрашивает: Укажите дату перелёта. Например: 10.04.2026 (Примчение: Бот осуществляет поиск только билетов в одну сторону. Для обратного билета повторите поиск.)
6. Пользователь вводит дату. 

Бот инициирует фактическое открытие браузера на сервере. И переход по следующей ссылки с подстановкой нужных переменных для поиска нужного предложения:

https://www.aeroflot.ru/sb/app/ru-ru#/search?adults=1&award=Y&cabin=business&children=0&childrenaward=0&childrenfrgn=0&infants=0&routes=MOW.20251210.SSH

routes - подставляем код из файла City_codes.py 
дату так же подставляем из тех данных которые ввёл пользователь ранее. Обрати внимание в ссылке дата в другом формате. Надо переконвертировать из ДД.ММ.ГГГГ в ГГГГММДД

Далее бот нажимает на кнопку "найти" <a role="button" tabindex="0" class="button button--wide button--lg  ">Найти</a>

Открывается выдача результата 

Нам необходимо сделать скриншот всего блока <div class="flight-search__panel-info"></div> и отправить пользователю.

Далее нам необходимо выполнить рассчет стоимости. Делаем это так:

Каждый рейс, который нам показывает сайт это class="flight-search" для высчитывания стоимости необходим нажать class="button button--wide button--outline button--disabled" и далее запомнить количество требуемых миль + сборы у тарифа Стандарт. Нас интересует только тариф стандарт.

Далее Нам надо сообщить пользователю стоимость билета за мили, которые мы только что просмотрели и запомнили. Выглядить это должно так:

Полная стоимость билета в 20:15 60 000 миль + 14 980 миль = 74 980 руб // это пример для тебя 
И так по каждому доступному рейсу в этот день. Не надо перегружать информацией о рейсе, достаточно только время вылета - мы уже прислали пользователю скриншот с исчерпывающей информацией о рейсах.

Стоимость 1 мили нужно иметь возможность задавать администратору. Возможно пока просто в текстовом документе. По умолчанию сейчас ставим 1 миля = 1 рубль

Если нет ни одного доступного рейса - во всех рейсах class="flight-search" указано "Билетов класса Бизнес нет в наличии", значит сообщаем пользователю об этом и просим выбрать другую дату.

Так же предусмотри возможность использовать Socks5 proxy для браузера, чтобы не получить бан по IP адресу от Аэрофлота
